# Computer Science & AI 8+ Workshops

These workshops complement the LEGO Education Computer Science & AI 8+ lesson plans. Each LEGO unit has its own workshop sequence.

Each plan supports two modalities:

- **2-hour workshop:** Combine lessons 1-2 in Hour 1 and lessons 3-4 in Hour 2.
- **4-hour workshop:** Run the 2-hour core, take a lunch break, then run the two-part lesson 5 extension.

The lunch break is separate from the four instructional hours. The core workshop always ends with a functioning project and a short demonstration. The extension adds deeper testing, creative improvements, and a team challenge.

## Shared Delivery Notes

- Plan for teams of four students.
- Prepare one kit and one compatible device for each team.
- Test a minimal working example before the workshop.
- Display the core success criteria during the build.
- Keep the final 10 minutes of each session protected for demonstration and reflection.
- For the 4-hour modality, schedule lunch only after every team has reached the core-project checkpoint.
